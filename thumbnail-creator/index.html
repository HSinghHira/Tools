About

Hello, My name is Harman Singh Hira, and I hail from the beautiful island of New Zealand. I am a part-time web developer, tech blogger, and a full-time husband. I modify this tool called Thumbnail Creator to help with the creation of thumbnails for blogs and YouTube videos.

The original tool was developed by Yasya El Hakim, and while I was impressed with it, I saw an opportunity to improve and expand its features. One crucial feature I needed was the ability to upload custom background images for thumbnails. This inspired me to redesign the tool using the Tailwind CSS framework. Along with the custom background image upload, I've added several new features to enhance its functionality.

I have plans to continue adding new features, so be sure to check the changelog for the latest updates.

I would like to extend my gratitude to Yasya for the initial backend script. Additionally, I am utilizing Tailwind CSS for the framework, HatScripts for flag images, and SimpleLocalize for website translation, so a big thanks to them too.

If you'd like to help translate this webpage into your local language, please reach out to me on Discord.

Thank you very much!
